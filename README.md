# Simple LDO Converter (5V → 3.3V)

A compact and efficient **5V to 3.3V Low Dropout (LDO) regulator PCB** designed using Altium Designer. This board provides stable and low-noise power for embedded and electronics applications.

---

##  Features
- Converts **+5V to +3.3V** using LDO regulator
- Low-noise and stable output
- Compact PCB layout
- Suitable for embedded systems and sensor modules
- Input/Output via standard connectors

---

## Tools Used
- Altium Designer
- PCB Layout & Routing
- Schematic Design
